A SIMPLE CALCULATOR

*KeyBoard Functionality


Instructions:

  1. 1. To use square root button from KeyBoard, use key 's'.
  2. 2. To use power, press key 'p' on keyboard.
  3. 3. To delete single character on screen, use backspace key.
  4. 4. To reset calculator screen, use Delete key on keyboard.
  5. 5. The calculator has the precision of up to 8 decimal places.
  6. 6. The calculate press Enter.
