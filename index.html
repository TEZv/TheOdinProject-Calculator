A SIMPLE CALCULATOR

/*Keyboard Functionality*/


Instructions:

  1. To use square root button from Keyboard, use key 'S'.
  2. To use power, press key 'P' on keyboard.
  3. To delete single character on screen, use Backspace key.
  4. To reset calculator screen, use Delete key on keyboard.
  5. The calculator has the precision of up to 8 decimal places.
  6. The calculate press Enter.
